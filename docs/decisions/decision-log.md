## 7. 인프라 현황 참고 (모델링 중엔 건드리지 않음)

- 리전 `asia-northeast3`(서울) 고정. VM `popcorn-app`(e2-small), Cloud SQL `popcorn-db`(PostgreSQL 16). "붙는다(`SELECT 1`)"까지만 검증됨.
